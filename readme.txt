### PopUp Anyway

Add PopUp advertisement easily, it is not affected with any AdBlock! and compatible with all major browsers like Google Chrome, Firefox, IE, Opera, phone and tablet.

### The Features

1. Easy to use, two options only.
2. It is not affected with any AdBlock.
3. Compatible with all major browsers.
4. Using PHP Cookie.

== Installation ==

### Installation

1. Upload 'popup-anyway' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WordPress General Settings page.
4. In "PopUp Link" field, enter your PopUp advertisement link (required option).
5. And in "PopUp Time" field, enter PopUp cookie time, default is 24 hours, will be display your popup ad every 24 hours per visitor (optional option).

### Note

In "PopUp Time" field, default time is 24 hours, If you change number, wait 24 hours to update option, because default time is 24 hours, for example if you change popup time to 2 hours, wait 24 hours to update option, and after 24 hours, if you change popup time from 2 hours to for example 48 hours, wait 2 hours to update option.
